color value options:

RED GREEN BLUE (RGB) VALUE declaration (property: color) & (value: rgb(0,128,0)) for the color "green" color: rgb(0,128,0)

HEX CODE VALUE declaration (property: color) & (value: #00008B) for the color "dark blue" color: #00008B

RGBA VALUE (added opacity) declaration (property: color) & (value: rgba(75,0,130,0.25)) for the color "indigo" with 25% opacity color: rgba(75,0,130,0.25)

HUE SATURATION LIGHTNESS ALPHA (HSLA) CODE VALUE declaration (property: color) & (value: hsla(0,100%,50%,0.75)) for the color "red" with 75% opacity color: hsla(0,100%,50%,0.75)
