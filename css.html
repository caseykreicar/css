HTML5 Layout:



Before HTML5 it was common practice to structure the web page using div element using classes and ids in many ways that could be cumbersome for others to interpret (not as clear code).

However, HTML5 added many structural elements to move away from that practice..(although not entirely) ie elements like <header>, <hgroup>, <footer>, <navigation>, <aside>, and <article> were added to give more emphasis on type of content relative to a web page's structural layout.
